# Documentation

This directory serves two public purposes:

- [Product demo workflow](DEMO_WORKFLOW.md) — the complete V1 and V2 presentation sequence, demonstration scenarios, suggested analysis questions and screenshot index.
- `assets/` — the full-page and selected dashboard screenshots used by the repository documentation.

The remaining HTML, CSS, JavaScript and JSON files are the static GitHub Pages deployment for the hosted dashboards. V1 is served from this directory root and V2 from `v2/`.

Development plans and superseded design contracts are intentionally excluded from the public documentation.
